## Project Description
This project is a secure healthcare web application developed using Flask and SQLite. It allows users to register, log in, and manage patient records.

The system was designed with basic security features to protect sensitive patient data.

## Features
- User registration and login
- Patient record management (Add, View, Edit, Delete)
- Session management
- Input validation

## Security Features
- Authentication system (login required)
- Password hashing
- Session control
- Restricted access to authorised users

## Testing
The system was tested using both successful and failed test cases. This includes login validation, access control, and patient record operations. Evidence of testing is included in the report.

## How to Run the System
1. Open terminal  
2. Navigate to the project folder  
3. Activate virtual environment  
4. Run the application using:  
   py app.py
